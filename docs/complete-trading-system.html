4. V1 Indicators — Detailed

V1 uses 6 indicators. Four are shared with V3 (covered in Section 6). Two are unique to V1:

V1-Unique: RSI (14-period) — Relative Strength Index

What It Measures

Speed and magnitude of recent price changes on a 0-100 scale. Tells you if buyers or sellers are in control.

Step 1: gain = max(price_change, 0) loss = abs(min(price_change, 0)) Step 2: avg_gain = EMA(gains, 14) avg_loss = EMA(losses, 14) Step 3: RS = avg_gain / avg_loss Step 4: RSI = 100 - (100 / (1 + RS)) Vote: RSI > 55 → BULL (+1 vote) RSI < 45 → BEAR (+0 votes) RSI 45-55 → NEUTRAL (+0.5 votes)

Three Uses in the System

VoterContributes to trend direction (V1 only)
Gate 2Block CE when RSI > 75 (overbought)
Gate 2bBlock all when RSI 48-57 (dead zone)

OVERSOLD < 20

Real data finding: RSI was NEUTRAL 28% of the time. This 0.5 vote padding actually helps V1 — it makes the 60% bull threshold easier to reach, letting profitable CE trades through.

V1-Unique: Supertrend (7, 2.5)

What It Measures

A volatility-based trend line that flips above (bearish) or below (bullish) the price. Built on ATR — adapts to market volatility.

ATR = Average True Range (7 periods) Upper Band = (High+Low)/2 + 2.5 x ATR Lower Band = (High+Low)/2 - 2.5 x ATR Direction: Price closes above Upper → BULL (line = lower) Price closes below Lower → BEAR (line = upper) Vote: Direction = 1 → BULL (+1 vote) Direction = -1 → BEAR (+0 votes) Parameters: period=7, multiplier=2.5 (Faster than default 10/3.0 for intraday)

Known Limitation

On Jan-Mar 2026 real data, Supertrend said BULL on 100% of candles. It never flipped bearish because NIFTY's decline was gradual, not sharp enough to break the ATR bands. This is why V3 replaced it.

Supertrend — Price vs Dynamic Line
BULL: Price ABOVE line
BEAR: Price BELOW line
Why it still works: The permanent BULL bias accidentally aligns with NIFTY's long-term upward drift. CE trades (which need BULL agreement) get an easy +1 vote, increasing signal generation for the most profitable trade type.

5. V3 Indicators — Detailed

V3 replaces RSI and Supertrend with two higher-scoring indicators:

V3-Unique: Linear Regression Slope (20-period) — Score: 88.5/100

What It Measures

The mathematical slope of a best-fit line through the last 20 price points. Positive slope = uptrend, negative = downtrend. Zero lag, never gets stuck.

For each candle: x = [0, 1, 2, ..., 19] (candle positions) y = [close prices for last 20 candles] slope = numpy.polyfit(x, y, degree=1)[0] Vote: slope > 0 → BULL (+1 vote) slope < 0 → BEAR (+0 votes) slope = 0 → NEUTRAL (+0.5 votes) Why it's #1 ranked (88.5/100): Balance: 46% bull / 54% bear (near perfect) Alignment: 87% agreement with actual direction Stability: 3.7 flips/day (very stable) Lead: 82% accuracy predicting 3 bars ahead No neutral: 99.9% always gives direction

Key advantage: Unlike Supertrend which was stuck BULL for 3 months, LinReg detected 54% bearish periods correctly. It's mathematically impossible for it to get stuck — the slope changes with every new candle.

V3-Unique: Awesome Oscillator (5/34) — Score: 87.5/100

What It Measures

Momentum using the difference between 5-period and 34-period Simple Moving Averages of the median price (High+Low)/2. Positive = bullish momentum accelerating, negative = bearish.

median_price = (High + Low) / 2 AO = SMA(median_price, 5) - SMA(median_price, 34) Vote: AO > 0 → BULL (+1 vote) AO < 0 → BEAR (+0 votes) Why it's #2 ranked (87.5/100): Balance: 44% bull / 56% bear (balanced) Alignment: 87% agreement with actual direction Stability: 3.3 flips/day (most stable) Lead: 80% accuracy 3 bars ahead No neutral: 100% always gives direction
Awesome Oscillator — Momentum Bars

AO < 0 = BEAR
vs MACD: AO uses median price (not close) and SMA (not EMA), making it smoother and less prone to whipsaws. It's like a calmer version of MACD that filters out intraday noise.

8. Trading Strategies: Momentum & OI Reversal

Strategy 1: Momentum Breakout

Buy options in the direction of confirmed trend when a high-scoring, liquid option with volume > 10,000 is found.

Entry: Score ≥ 50, Volume > 10K, Premium 50-500 SL: ATR-based (multiplier 1.5, floor 15%) Target: Entry + (risk x 2.0 R:R) Result: 20% WR on real data (weak)

Strategy 2: OI Reversal

Buy options at strikes where Open Interest confirms support/resistance. High Put OI = support → buy CE. High Call OI = resistance → buy PE.

Entry: OI > 200K at nearby strike confirms level SL: Same ATR-based calculation Target: Same R:R ratio Result: 68% WR on real data (dominant strategy)
OI Reversal is the primary profit generator across all backtesting. It generated +79K in Nov 2025 alone (82% WR). Momentum strategy consistently underperforms — consider removing it entirely.

4-Stage Trailing Stop Loss

move_pct = (Current - Entry) / (Target - Entry) x 100 Stage 1: move ≥ 20% → SL = Entry (breakeven, risk-free) Stage 2: move ≥ 50% → SL = Entry + 30% of profit Stage 3: move ≥ 70% → SL = Entry + 50% of profit Stage 4: move ≥ 85% → SL = Entry + 70% of profit Example (Entry=200, SL=170, Target=260): Price 212 (20%): SL 170 → 200 (breakeven) Price 230 (50%): SL → 209 Price 242 (70%): SL → 221 Price 251 (85%): SL → 236
